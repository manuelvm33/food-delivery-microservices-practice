# food-delivery-microservices-practice
🎓 Microservices food delivery platform - Practice project implementing Spring Boot, Angular, Jenkins CI/CD, Docker

## 📋 About

This is a practice project developed as part of a Udemy course on microservices architecture. 
It demonstrates the implementation of a complete food delivery application using modern 
technologies and best practices.

**⚠️ Note**: This is a learning project, not intended for production use.

## 🏗️ Architecture

Microservices-based architecture implementing:
- **Backend**: Java 17, Spring Boot, Spring Cloud
- **Frontend**: Angular/React [tu framework]
- **CI/CD**: Jenkins
- **Testing**: JUnit 5, Integration Tests
- **Containerization**: Docker, Docker Compose
- **Service Discovery**: Eureka Server
- **API Gateway**: Spring Cloud Gateway

## 🚀 Services

- **User Service**: Authentication and user management
- **Order Service**: Order processing and management
- **Restaurant Service**: Restaurant and menu management
- **Delivery Service**: Delivery tracking and logistics
- **Payment Service**: Payment processing

## 🛠️ Technologies

- Java 17
- Spring Boot 3.x
- Spring Cloud
- Maven
- JUnit 5
- Docker
- Jenkins
- PostgreSQL/MySQL
- MongoDB (if applicable)

## 📚 Learning Objectives
